Image block layout

When you tap + → Image, how should a single or multiple images display on the page?

🌅

A — Full width

Each image stretches edge-to-edge. Clean, bold. If you pick multiple photos they stack vertically.

B — Auto grid (1 → full, 2+ → 2-column)

Single image is full width. Picking 2+ images auto-arranges into a 2-column grid. Compact and gallery-like.

C — Smart collage (1 full, 2 side-by-side, 3+ collage)
