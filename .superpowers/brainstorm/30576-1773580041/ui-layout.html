Key layout details

  • Page background: Canvas drawLine every 28dp (blue, α 0.35) + one vertical red line at 40dp from left (α 0.4). Drawn behind all blocks.
  • Content inset: 48dp left padding (right of margin line), 16dp right. Title sits above first block with a thin separator.
  • "+" button: 20dp circle, positioned in the left margin (8–10dp from edge), appears on every block row. Tapping opens a modal bottom sheet with 4 block types.
  • Formatting bubble: Popup anchored to the focused RichTextEditor; shows B / I / U / bullet. Dismissed on tap-outside or cursor collapse.
  • Image block: 1 image = full width rounded card; 2+ = 2-column grid. Tap-and-hold on image to delete.
  • Checklist: Each row has a Checkbox + BasicTextField. Checked items get strikethrough. Enter key appends a new item.
  • Divider: Faded horizontal gradient line (transparent → #aaa → transparent).
